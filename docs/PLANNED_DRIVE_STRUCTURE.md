## 1. 조직 및 업무 컨텍스트

- **기관 성격**: 비영리 민간단체, 성인 대상 평생교육기관
- **조직 단위**: 총회 → 운영위원회 (동아리, 강사협의회, 정회원, 사무처)
- **학기 구조**: 연 4학기 (겨울 1~3월 / 봄 4~6월 / 여름 7~9월 / 가을 10~12월)
- **강사 처우**: 강사비 없음 → 성남시자원봉사센터에 자원봉사 시간 등록
- **외부 시스템**: 배움숲(수강신청), 네이버 카페(홍보), 은행(거래내역)
- **Google Workspace**: 사용 중 → Shared Drive로 이전 예정

       01_거버넌스 ~ 99_보관 전체 폴더 트리 생성

3단계  파일 이동 (우선순위 순)
       ① 01_거버넌스 (정관, 총회 등 핵심 문서)
       ② 02_학사운영 (가장 많은 파일)

       ④ 04_회계
       ⑤ 05_연간행사 + 06_사업
       ⑥ 07_동아리 + 08_홍보·미디어
       ⑦ 09_공통자산 + 99_보관

4단계  중복 파일 정리
       정관, 현수막 등 2곳에 있는 파일 확인 후 하나로 통합

5단계  검토 및 마무리
       빈 폴더 정리, 누락 파일 확인
