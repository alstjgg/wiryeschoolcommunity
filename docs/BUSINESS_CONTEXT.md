## 1. 기관 개요

### 1.1 위례인생학교란?

위례인생학교는 성인을 대상으로 하는 평생교육 프로그램입니다. 다양한 분야의 강좌를 제공하며, 지역 주민들의 평생 학습을 지원합니다.

### 1.2 운영 주체

- **운영**: 위례인생학교 사무처
- **플랫폼**: 배움숲 (수강신청 포털)
- **홍보**: 네이버 카페

---

## 2. 학기 구조

### 2.1 학기제

| 학기 | 시작월 | 기간 |
|:---:|:-----:|:----:|
| 1학기 | 1월 | 1월 ~ 3월 |
| 2학기 | 4월 | 4월 ~ 6월 |
| 3학기 | 7월 | 7월 ~ 9월 |
| 4학기 | 10월 | 10월 ~ 12월 |

### 2.2 학기 시작일

- **원칙**: 각 월의 첫 주
- **예외**: 강사 사정에 따라 최대 1주 지연 가능
- **참고**: 모든 수업은 요일이 다르지만 첫 주에 개강

### 2.3 주요 일정

| 시점 | 업무 |
|-----|------|
| 학기 시작 2달 전 | 강사에게 강의 계획서 요청 |
| 학기 시작 1달 전 | 강의 계획서 검토, 배움숲/카페 홍보 게시 |
| 전월 말일까지 | 수강신청 접수 |
| 개강 첫 주 전 토요일 | 강사에게 수강자 목록 전달 |
| 개강 1주 후 | 환불 마감 |
| 학기 종료 후 | 운영 보고서 작성 |

## 3. 회원 제도

### 3.1 회원 유형

| 유형 | 가입비 | 유효 기간 | 특징 |
|-----|:-----:|----------|------|
| **일반회원** | 1만원 | 해당 학기 | 수강료 입금 확인 후 등록 처리 |
| **연회원** | 12만원 | 당년 3월 ~ 후년 2월 (최대 4학기) | 입금 확인 없이 등록 처리 가능 |

### 3.2 연회원 처리 방식

연회원은 수강료 입금 확인 없이 수강신청을 "등록" 상태로 변경할 수 있습니다.

**처리 흐름**:
1. 수강신청 접수 확인
2. 회원 정보에서 연회원 여부 확인
3. 연회원인 경우: 입금 확인 없이 "등록" 처리
4. 일반회원인 경우: 입금 내역 확인 후 "등록" 처리

### 3.3 신규 가입

신규 가입 시 가입비(1만원)가 필요합니다.

**입금 패턴**:
- 가입비 + 수강료 별도 입금
- 가입비 + 수강료 합산 입금

## 4. 수강 등록

### 4.1 수강료

- **기본 수강료**: 2만원 (전 과목 동일)
- **참고**: 향후 과목별 차등 가능성 있음

### 4.2 입금 케이스

| 케이스 | 설명 | 입금 형식 예시 |
|-------|------|--------------|
| 기본 | 단일 과목 수강 | "경제해설_홍길동" |
| 신규 (별도) | 가입비 + 수강료 각각 | "가입비_홍길동", "경제해설_홍길동" |
| 신규 (합산) | 가입비 + 수강료 합산 | 3만원 입금 |
| 다과목 (별도) | 과목별 각각 입금 | "경제해설_홍길동", "영어회화_홍길동" |
| 다과목 (합산) | 여러 과목 합산 입금 | 4만원 입금 |
| 연회원 | 연회비 납부 | 12만원 입금 |
| 중복 | 동일 건 중복 입금 | 환불 처리 필요 |

### 4.3 등록 상태

| 상태 | 설명 |
|:---:|------|
| ✅ 등록완료 | 입금 확인 완료 또는 연회원 |
| 🔶 확인필요 | 금액 기반 추정 (수동 확인 필요) |
| ⚠️ 이름불일치 | 입금자명과 신청자명 불일치 |
| ❌ 미입금 | 입금 내역 없음 |
| 🔄 중복입금 | 동일 건 중복 입금 |

---

## 5. 환불 규칙

### 5.1 기본 규칙

- **개강 전**: 전액 환불 가능
- **개강 1주 후**: 환불 불가
- **공지**: 개강 전 환불 규정 안내

### 5.2 환불 처리

환불은 별도 민원 담당자가 처리합니다.

---

## 6. 강의 관리

### 6.1 강의 계획서

강사가 매 학기 강의 계획서를 제출합니다.

**포함 내용**:
- 강좌명
- 강사 정보
- 강의 개요
- 주차별 계획
- 준비물 등

**주의사항**:
- 과목/강사는 학기마다 변동 가능
- 계획서를 통해 해당 학기 과목 정보 확인

### 6.2 출석 관리

**현재 방식**: 종이 출석부
- 학생 이름 목록
- 출석 시 사인 또는 표시
- 강사가 관리

**향후 계획**: 종이 출석부 촬영 → AI 인식 → 디지털화

### 6.3 휴강/보강

- 강사가 직접 처리
- 처리 후 사무처에 알림

---

## 7. 수강생 관리

### 7.1 수강생 문의

민원 담당자가 처리합니다.

### 7.2 관리 대상 식별 (블랙리스트)

명확한 기준은 없으나, 다음 정보를 기록하여 관리 참고:
- 수강 취소 이력
- 출석률
- 민원 이력

---

## 8. 외부 시스템

### 8.1 배움숲

수강신청 포털입니다.

**주요 기능**:
- 강좌 목록 게시
- 수강신청 접수
- 신청 상태 관리 ("신청" → "등록")

**관리자 작업**:
- 강좌 정보 입력
- 신청자 입금 확인 후 상태 변경

### 8.2 네이버 카페

홍보 채널입니다.

**게시 내용**:
- 학기별 강좌 안내
- 수강신청 안내
- 공지사항

---

## 9. 용어 정리

| 용어 | 설명 |
|-----|------|
| 배움숲 | 수강신청 포털 사이트 |
| 연회원 | 1년 회비를 납부한 회원 (최대 4학기 수강 가능) |
| 일반회원 | 학기별 수강료를 납부하는 회원 |
| 가입비 | 신규 회원 가입 시 납부하는 비용 (1만원) |
| 수강료 | 강좌 수강을 위해 납부하는 비용 (현재 2만원) |
| 개강 | 해당 학기 첫 수업 시작 |
| 수강신청현황 | 배움숲에서 관리하는 신청자 목록 |
